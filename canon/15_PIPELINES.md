## What each pipeline's workflow looks like (short)

1. **Blender all-in-one** — Geo Nodes city → Megascans/PolyHaven materials → sim nodes → Cycles render → compositor. One tool, free, you already started here.
2. **Blender → UE5** — model/scatter in Blender, export, light & render real-time in UE5 (Lumen GI, Nanite detail). Best balance for dynamic flythroughs.
3. **UE5 native + PCG** — build the whole city with UE5 Procedural Content Generation + Megascans, real-time. Steeper but cutting-edge.
4. **Houdini + UE5** — Houdini procedurally generates city + sims (the real industry route), render in UE5/Karma. Highest ceiling, hardest.
5. **Cinema 4D + Octane** — likely the reference reel's family: motion-design + GPU render, fast iteration, gorgeous.
6. **3ds Max + V-Ray** — photoreal archviz stills/slow shots; weaker for dynamic sims.
7. **Unity HDRP** — UE5 alternative; great real-time, smaller asset ecosystem.
8. **CityEngine → UE5** — rule-based city generation (perfect for OORIKH's grid+rings), then dress in UE5.
9. **Twinmotion** — drop models in, instant photoreal real-time; less fine control.
10. **Gaea/World Creator** — pro terrain/erosion, export heightmaps to your main tool.
11. **ZBrush + Substance** — sculpt hero palace/gates, texture in Substance; import to Blender.
12. **RealityCapture** — photoscan real rocks/walls/props → ultimate-realism assets.
13. **SpeedTree** — best-in-class trees/palms for gardens.
14. **Marvelous Designer** — simulate real cloth banners, sails, awnings.
15. **Blender + Substance Painter** — Blender for everything + Substance for film-grade materials (the realism multiplier).
